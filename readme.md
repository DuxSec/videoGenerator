## DEVELOPER SETUP

Install ImageMagick from source; ./imageMagicksInstaller or https://imagemagick.org/script/download.php

Install requirements.txt

Normal pip

```pip install -r requirements.txt```

Pip3:

```pip3 install -r requirements.txt```
